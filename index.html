Frequently Asked Questions

Are you curious like everyone else?
Here are some FAQs by our course subscribers.

Flexbox is a one-dimensional layout model, it can either work in rows or columns at a time.Flexbox is content based. Grid is a two-dimensional layout model that works in both rows and columns. Grid is container based and is more focused on layout.

Tailwind CSS is a popular utility-first framework that can be used to create dynamic web designs.Tailwind is a highly customizable compared to Bootstrap. Bootstrap is a mobile-first framework with sets of mobile-friendly pre-set components to use.

Every element in a web page is a box. The CSS box model is used to determine how a box should appear, how it should be positioned on a webpage. Every box consists of content areas, padding, border, margin. When we calculate the width and height of an element with CSS, we just calculate the content area. To calculate the full size, we should also add padding, borders and margins.

Semantic HTML uses elements for their given purpose, it's easier for both developers and machines to read and understand it. Semantic elements also helps with SEO (Search Engine Optimization). It improves the code structure and make code more readable. (for example: header footer article etc)
